# **App Name**: OTW

## Core Features:

- Order Availability: Display a live list of available delivery orders, including origin and destination addresses.
- Route Visualization: Display assigned orders with details. Show the shortest route for a single order or an optimized multi-stop route for multiple orders, differentiating routes based on vehicle type (car/bike) when routing rules apply.
- Delivery Optimization: Suggest grouping multiple delivery items into an optimized trip using an advanced routing algorithm to calculate the most efficient path for multiple pickups and drop-offs, accounting for vehicle type.
- Status Updates: Allow couriers to update the status of each order in real-time.
- AI Route Optimization: Generative AI tool recommends optimized delivery routes to minimize total travel distance for multiple deliveries.

## Style Guidelines:

- Primary color: Teal (#008080) for a sense of reliability and efficiency.
- Background color: Light teal (#E0F8F8). This creates a clean, professional backdrop without distracting from the app's main features.
- Accent color: Soft orange (#FFA07A) for highlights, call-to-action buttons, and important notifications. This provides a warm, inviting contrast.
- Body and headline font: 'PT Sans', sans-serif, for clear readability on various screen sizes.
- Use clear and concise icons to represent order status, locations, and vehicle types. Icons should be easily recognizable and consistent throughout the app.
- Maintain a clean and intuitive layout, prioritizing key information such as order details and route visualization. Utilize a card-based design for order listings.
- It should be mobile first
- Implement subtle animations to provide feedback during interactions, such as route updates and status changes. Keep animations minimal and purposeful to enhance user experience without overwhelming.
